# Customer Churn Prediction using Neural Network

This project predicts whether a telecom customer will churn using a trained Neural Network model built with TensorFlow.

## Dataset
IBM Telco Customer Churn Dataset:
https://raw.githubusercontent.com/IBM/telco-customer-churn-on-icp4d/master/data/Telco-Customer-Churn.csv

## Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Gradio

## Features Used
- OnlineBackup
- PaperlessBilling
- PaymentMethod
- MonthlyCharges
- TotalCharges

```bash
pip install -r requirements.txt
